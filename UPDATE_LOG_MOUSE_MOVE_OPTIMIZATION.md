# 更新日志 - 鼠标移动手势优化

## 版本信息
**版本号**: v4.3
**更新日期**: 2026-02-13
**主要变更**: 优化鼠标移动手势识别逻辑，仅需食指中指伸出并靠拢即可触发

## 变更详情

### 🔥 核心功能优化
- **鼠标移动触发条件简化**: 从"食指中指伸直并靠拢，无名指小指也伸直"改为"仅需食指中指伸出并靠拢"
- **手势自然度提升**: 用户无需刻意伸直所有手指，使操作更加自然舒适
- **识别准确率改善**: 简化判断逻辑，减少误触发的可能性

### ✨ 技术实现

#### 手势识别器修改 (`recognition/gesture_recognizer.py`)
- 简化鼠标移动手势判断条件
- 移除了对无名指和小指伸直状态的检查
- 保持原有的距离阈值和稳定性机制

```python
# 修改前
elif (index_extended and middle_extended and 
      index_middle_tip_distance < self.thresholds['finger_proximity'] and
      ring_extended and pinky_extended):
    # 食指和中指伸直并靠拢，其他手指伸直 = 鼠标移动
    return "鼠标移动"

# 修改后  
elif (index_extended and middle_extended and 
      index_middle_tip_distance < self.thresholds['finger_proximity']):
    # 食指和中指伸直并靠拢 = 鼠标移动
    # 不再要求无名指和小指也伸直
    return "鼠标移动"
```
